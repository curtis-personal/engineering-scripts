## Lists

### Unordered - Bulletpoint
* Item 1
* Item 2
* Item 2a
* Item 2b

### Unordered - Checklist
- [ ] Item 1
- [X] Item 2 - Complete
- [ ] Item 2a
- [ ] Item 2b

### Ordered 
1. Item 1
2. Item 2
3. Item 3
   1. Item 3a
   2. Item 3b
